# Requirements

## [Neovim 0.11](https://github.com/neovim/neovim/releases/tag/stable)

- The neweste version often is not in the repo, so use: <br>
    `sudo add-apt-repository ppa:neovim-ppa/stable` <br>
    `sudo apt update` <br>
    `sudo apt install neovim`
